Upload Guidelines

Images

  • Supported formats: JPG, PNG, GIF, BMP
  • Organize in folders by class (e.g., /cats, /dogs)
  • Recommended: 100+ images per class
  • Image size: Any (will be resized to 224x224)

Text

  • Supported formats: TXT, CSV, JSON
  • CSV should have 'text' and 'label' columns
  • Recommended: 200+ samples per class
  • UTF-8 encoding preferred

CSV/Tabular

  • First row should be column headers
  • Target column should be named 'target' or 'label'
  • Numerical and categorical columns supported
  • Missing values will be handled automatically
